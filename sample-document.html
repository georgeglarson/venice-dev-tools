Venice AI SDK Documentation

Introduction

The Venice AI SDK provides a comprehensive set of tools for interacting with the Venice AI API. This document outlines the key features and capabilities of the SDK, with a focus on character interactions and document analysis.

Character Interactions

One of the most powerful features of the Venice AI SDK is the ability to interact with predefined AI characters. These characters have unique personalities, knowledge bases, and conversational styles, making them ideal for specialized use cases.

Characters can be used in various contexts, including:

  • Customer service chatbots
  • Educational assistants
  • Entertainment applications
  • Specialized domain experts

Document Analysis

The SDK now supports document analysis capabilities, allowing characters to analyze and interpret various file types, including:

  • PDF documents
  • Images (JPG, PNG, GIF, WebP)
  • Vector graphics (SVG)

This feature enables powerful use cases such as:

  • Extracting information from business documents
  • Analyzing charts and graphs
  • Interpreting technical diagrams
  • Summarizing research papers

Code Example


// Analyze a PDF with a character
const fs = require('fs');
const pdfBuffer = fs.readFileSync('document.pdf');
const base64Pdf = pdfBuffer.toString('base64');

const response = await venice.chat.completions.create({
  model: "qwen-2.5-vl",
  messages: [
    { 
      role: "user", 
      content: [
        { type: "text", text: "Summarize this document" },
        { type: "image_url", image_url: { url: `data:application/pdf;base64,${base64Pdf}` } }
      ] 
    }
  ],
  venice_parameters: {
    character_slug: "professor"
  }
});

Conclusion

The Venice AI SDK continues to evolve with new features and capabilities. The addition of document analysis to character interactions represents a significant enhancement to the platform, enabling more sophisticated AI applications across various domains.
